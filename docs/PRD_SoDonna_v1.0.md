## 1. 产品概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 产品名称 | **SoDonna** |
| Slogan | 你负责思考商业版图，剩下的交给我。 |
| 产品形态 | Web App（响应式，桌面端优先） |
| 定位 | 专为"一人公司"CEO 打造的终极支持系统 |

### 1.2 核心理念——"1.1 公司"

在 AI 时代，一个人就是一支军队。但最聪明的将军，从不自己去磨刀。

- **1** = 你：独一无二的战略眼光、创意和商业直觉
- **0.1** = SoDonna：一套不知疲倦、没有情绪、永远在线的支持系统

我们不干涉你的决断，我们只负责让你的每一个决断都丝滑落地。

### 1.3 目标用户

一人公司 CEO / 超级个体——独自掌控业务全局，兼任战略、执行、运营多重角色的创始人。

## 2. 用户画像与痛点分析

### 2.1 典型用户画像

#### Persona A：独立电商创始人

| 维度 | 描述 |
|------|------|
| 姓名 | 小陈 |
| 年龄 | 28 岁 |
| 背景 | 从大厂出走，自建 DTC 品牌，独自运营 Shopify 店铺 |
| 日常 | 选品、投放、客服、发货、内容创作一人包揽 |
| 痛点 | 每天在 10+ 个后台之间来回切换，灵感转瞬即逝无处记录 |
| 目标 | 用更少的时间管理工具，用更多的时间做产品和品牌 |

#### Persona B：自由咨询顾问

| 维度 | 描述 |
|------|------|
| 姓名 | David |
| 年龄 | 35 岁 |
| 背景 | 前 McKinsey 咨询师，独立执业，服务 5-8 个长期客户 |
| 日常 | 客户会议、方案撰写、知识管理、财务记账 |
| 痛点 | 没有助理帮做会议纪要，上个月的洞察这个月找不到了 |
| 目标 | 让每次决策都有迹可循，方便复盘和复用 |

#### Persona C：内容创业者

| 维度 | 描述 |
|------|------|
| 姓名 | 阿杰 |
| 年龄 | 32 岁 |
| 背景 | 全职自媒体，YouTube + Newsletter + 付费社群 |
| 日常 | 选题、脚本、拍摄、剪辑、分发、社群运营 |
| 痛点 | 订阅了 Notion、Airtable、Zapier 等 8 个 SaaS，数据在 A 和 B 之间搬运消耗大量时间 |
| 目标 | 一个统一的指挥中心，5 分钟掌控全局 |

### 2.2 三大核心痛点

#### 痛点 1：精力碎片化

上一秒在构思宏大战略，下一秒却在寻找昨天随手记下的某个客户灵感。CEO 的心流被无穷无尽的琐碎执行打断，战略思考时间被压缩到接近于零。

#### 痛点 2：工具综合征

订阅了无数个 SaaS 软件，却每天要把一半的时间花在将数据从 A 搬运到 B 上。工具本该是放大器，却成了负担。

#### 痛点 3：决策无记录

没有人帮你做会议纪要，没有复盘日志，上个月的成功经验这个月无法复用。每一次决策都是从零开始，无法积累组织记忆。

### 2.3 用户旅程地图——一人公司 CEO 的一天

| 时间 | 场景 | 当前痛点 | SoDonna 解决方案 |
|------|------|----------|-----------------|
| 08:00 | 晨间规划 | 打开 5 个 App 看昨日数据，拼凑全局 | 指挥中心仪表盘：一屏掌控核心指标 |
| 09:00 | 灵感突发 | 语音备忘录录完就忘，找不回来 | 数字外脑：语音转文字 + 自动归档 + 标签 |
| 10:30 | 客户沟通 | 聊完没记录，下次开会忘了上次承诺 | 决策日志：自动关联客户 + 时间线 |
| 13:00 | 内容分发 | 手动复制粘贴到 3 个平台 | 工具流：一键分发 + 更新日志 |
| 15:00 | 数据复盘 | 在 Google Analytics、Stripe、邮件营销后台来回跳 | 仪表盘聚合：流量 / 转化 / 营收一目了然 |
| 18:00 | 日终总结 | 根本没有总结的习惯，因为太累了 | AI 复盘助手：自动生成当日摘要 + 明日建议 |

## 5. UX/UI 设计规范

> **设计灵感**：参考 [Future.co/pro](https://future.co/pro) 的高端视觉语言——深色沉浸式背景、毛玻璃层叠、大号表现力字体、精致光影动效。SoDonna 的视觉目标是：**让用户在打开页面的瞬间感受到"这就是为我准备的"。**

### 5.1 设计总纲——Dark Glassmorphism + Editorial Typography

| 维度 | 方向 |
|------|------|
| 基调 | **全页深色沉浸式**（Dark-first），不是传统的白底蓝字 SaaS 风格 |
| 质感 | **毛玻璃（Glassmorphism）**——半透明面板 + `backdrop-blur` + 微妙边框光 |
| 字体 | **大号 Display 字体做视觉冲击**，正文克制留白 |
| 光影 | **柔光渐变 Orb** 漂浮在背景中，制造空间感和呼吸感 |
| 动效 | **物理感运动**——弹性缓动（spring）、视差滚动、光晕跟随鼠标 |
| 情绪 | **高端、沉稳、自信**——Donna 的气场：她走进房间，不需要自我介绍 |

- **背景**：纯黑 `#09090B` 底色，叠加 2-3 个大尺寸径向渐变 Orb（紫 `#7C3AED` → 蓝 `#2563EB` → 青 `#06B6D4`），Orb 边缘做 80px 高斯模糊，`opacity: 0.3`，缓慢脉动呼吸（`scale 1.0 ↔ 1.05`, `duration: 8s`, `ease-in-out`, infinite）
- **Orb 交互**：鼠标移动时 Orb 中心缓慢跟随（`lerp 0.02`），制造空间感
- **主标题**：72-96px（Desktop），字体 `Instrument Serif` 或 `Playfair Display`（西文）+ `Noto Serif SC`（中文），`font-weight: 400`（衬线字体用正常字重更优雅），使用 `background-clip: text` 做白→淡紫渐变填充，字间距 `0.02-0.05em`
- **副标题**：20-22px，`rgba(255,255,255,0.6)`，`Satoshi` / `Noto Sans SC`，行高 1.7
- **CTA 按钮**：毛玻璃态——`background: rgba(255,255,255,0.08)`，`backdrop-filter: blur(12px)`，`border: 1px solid rgba(255,255,255,0.15)`，圆角 `999px`（全圆角胶囊），padding `16px 36px`，文字白色 16px，悬停时边框光增强至 `rgba(255,255,255,0.4)` + 内发光 `box-shadow: inset 0 0 20px rgba(255,255,255,0.05)`
- **布局**：100vh，文字居中，最大宽度 800px
- **滚动指示器**：底部居中，小圆点 + 向下箭头，opacity 脉动
