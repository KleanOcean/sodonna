## 1. 产品概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 产品名称 | **SoDonna** |
| Slogan | 你负责思考商业版图，剩下的交给我。 |
| 产品形态 | Web App（响应式，桌面端优先） |
| 定位 | 专为"一人公司"CEO 打造的终极支持系统 |

### 1.2 核心理念——"1.1 公司"

在 AI 时代，一个人就是一支军队。但最聪明的将军，从不自己去磨刀。

- **1** = 你：独一无二的战略眼光、创意和商业直觉
- **0.1** = SoDonna：一套不知疲倦、没有情绪、永远在线的支持系统

我们不干涉你的决断，我们只负责让你的每一个决断都丝滑落地。

### 1.3 目标用户

一人公司 CEO / 超级个体——独自掌控业务全局，兼任战略、执行、运营多重角色的创始人。

## 2. 用户画像与痛点分析

### 2.1 典型用户画像

#### Persona A：独立电商创始人

| 维度 | 描述 |
|------|------|
| 姓名 | 小陈 |
| 年龄 | 28 岁 |
| 背景 | 从大厂出走，自建 DTC 品牌，独自运营 Shopify 店铺 |
| 日常 | 选品、投放、客服、发货、内容创作一人包揽 |
| 痛点 | 每天在 10+ 个后台之间来回切换，灵感转瞬即逝无处记录 |
| 目标 | 用更少的时间管理工具，用更多的时间做产品和品牌 |

#### Persona B：自由咨询顾问

| 维度 | 描述 |
|------|------|
| 姓名 | David |
| 年龄 | 35 岁 |
| 背景 | 前 McKinsey 咨询师，独立执业，服务 5-8 个长期客户 |
| 日常 | 客户会议、方案撰写、知识管理、财务记账 |
| 痛点 | 没有助理帮做会议纪要，上个月的洞察这个月找不到了 |
| 目标 | 让每次决策都有迹可循，方便复盘和复用 |

#### Persona C：内容创业者

| 维度 | 描述 |
|------|------|
| 姓名 | 阿杰 |
| 年龄 | 32 岁 |
| 背景 | 全职自媒体，YouTube + Newsletter + 付费社群 |
| 日常 | 选题、脚本、拍摄、剪辑、分发、社群运营 |
| 痛点 | 订阅了 Notion、Airtable、Zapier 等 8 个 SaaS，数据在 A 和 B 之间搬运消耗大量时间 |
| 目标 | 一个统一的指挥中心，5 分钟掌控全局 |

### 2.2 三大核心痛点

#### 痛点 1：精力碎片化

上一秒在构思宏大战略，下一秒却在寻找昨天随手记下的某个客户灵感。CEO 的心流被无穷无尽的琐碎执行打断，战略思考时间被压缩到接近于零。

#### 痛点 2：工具综合征

订阅了无数个 SaaS 软件，却每天要把一半的时间花在将数据从 A 搬运到 B 上。工具本该是放大器，却成了负担。

#### 痛点 3：决策无记录

没有人帮你做会议纪要，没有复盘日志，上个月的成功经验这个月无法复用。每一次决策都是从零开始，无法积累组织记忆。

### 2.3 用户旅程地图——一人公司 CEO 的一天

| 时间 | 场景 | 当前痛点 | SoDonna 解决方案 |
|------|------|----------|-----------------|
| 08:00 | 晨间规划 | 打开 5 个 App 看昨日数据，拼凑全局 | 指挥中心仪表盘：一屏掌控核心指标 |
| 09:00 | 灵感突发 | 语音备忘录录完就忘，找不回来 | 数字外脑：语音转文字 + 自动归档 + 标签 |
| 10:30 | 客户沟通 | 聊完没记录，下次开会忘了上次承诺 | 决策日志：自动关联客户 + 时间线 |
| 13:00 | 内容分发 | 手动复制粘贴到 3 个平台 | 工具流：一键分发 + 更新日志 |
| 15:00 | 数据复盘 | 在 Google Analytics、Stripe、邮件营销后台来回跳 | 仪表盘聚合：流量 / 转化 / 营收一目了然 |
| 18:00 | 日终总结 | 根本没有总结的习惯，因为太累了 | AI 复盘助手：自动生成当日摘要 + 明日建议 |

## 3. 产品功能需求表

### P0——核心功能（MVP 必须）

| 编号 | 功能 | 描述 |
|------|------|------|
| F-001 | 数字外脑 — 灵感归档 | 用户可快速输入文字/语音备忘，系统自动归档 |
| F-002 | 数字外脑 — 智能标签 | 系统基于内容自动打标签（客户、项目、主题） |
| F-003 | 数字外脑 — 全局检索 | 支持关键词 / 标签 / 时间范围的全文检索 |
| F-004 | 指挥中心仪表盘 | 聚合核心业务指标（流量、转化、财务）的可视化面板 |
| F-005 | 候补名单 / 落地页 | 产品落地页 + 邮箱收集，用于验证需求和积累早期用户 |

### P1——增强功能（V1.1）

| 编号 | 功能 | 描述 |
|------|------|------|
| F-101 | 工具流自动化 — 内容分发 | 一键将内容分发到多个平台（社交媒体、Newsletter 等） |
| F-102 | 工具流自动化 — 操作日志 | 所有自动化操作留痕，可追溯、可撤销 |
| F-103 | 语音备忘转文字 | 支持语音输入，自动转录为结构化文字并归入外脑 |

### P2——进阶功能（V2.0）

| 编号 | 功能 | 描述 |
|------|------|------|
| F-201 | AI 决策复盘 | 基于历史决策日志，自动生成复盘报告和改进建议 |
| F-202 | 多工具集成 | 连接 Google Analytics、Stripe、Notion、Airtable 等第三方服务 |
| F-203 | 智能日报 | 每日自动汇总当日操作、灵感和指标变化，生成可读的日终报告 |

### 5.2 视觉风格

**设计调性**：专业 + 温暖 + 高效。不冷冰冰，但绝不花哨。像一位穿着得体的私人助理——给人信赖感。

#### 字体

| 层级 | 字体 | 字号 | 字重 |
|------|------|------|------|
| H1 主标题 | Inter / Noto Sans SC | 48-64px | Bold (700) |
| H2 副标题 | Inter / Noto Sans SC | 32-40px | Semibold (600) |
| H3 功能标题 | Inter / Noto Sans SC | 24-28px | Semibold (600) |
| Body 正文 | Inter / Noto Sans SC | 16-18px | Regular (400) |
| Caption 辅助 | Inter / Noto Sans SC | 13-14px | Regular (400) |
| 英文衬线点缀 | Playfair Display | 24-32px | Italic |

- 中文优先使用 Noto Sans SC，西文使用 Inter
- 行高：标题 1.2-1.3，正文 1.6-1.8
- 英文经典台词（如 Footer 区域）使用 Playfair Display 斜体，营造品牌质感

### 5.4 交互与动效

#### 滚动触发动效

| 元素 | 动效 | 参数 |
|------|------|------|
| 痛点卡片 | 从下方淡入上浮 | `translateY(24px) → 0`, `opacity: 0 → 1`, `duration: 600ms`, `ease-out` |

## 6. 品牌与文案调性

### 6.1 Donna 人格化设定

SoDonna 的名字和人格灵感来源于美剧《Suits》中的 Donna Paulsen——Harvey Specter 的完美秘书。

| 维度 | 特征 |
|------|------|
| 身份 | 你的首席执行助手（Chief of Staff） |
| 性格 | 自信、高效、不卑不亢、有一丝幽默感 |
| 能力 | 过目不忘、提前预判、无声执行 |
| 关系 | 不是老板，不是下属，是让你安心的搭档 |
| 口头禅 | "你还没开口，我已经安排妥当。" |

### 6.2 文案原则

| 原则 | 说明 | 正例 | 反例 |
|------|------|------|------|
| 自信 | Donna 从不说"也许"、"或许" | "我已经安排妥当。" | "我可以试试帮你安排。" |
| 简洁 | 一句话能说清楚绝不用两句 | "5 分钟，掌控全局。" | "我们的仪表盘可以帮助您在短时间内了解业务全貌。" |
| 有温度 | 专业但不冷漠，像一个真正关心你的搭档 | "你负责思考，剩下的交给我。" | "本平台提供全方位业务支持服务。" |
| 对等 | 不跪着做服务，是合伙人的语气 | "我永远比你提前三步。" | "我们竭诚为您服务。" |
